Inverse Optimisation for Topas

This repository integrates the open source bayesian optimisation code Bayesian Optimisation with the monte carlo radiation transport code TopasMC, hence enabling users to apply the power of formal mathematical optimisation with the well established use cases for monte carlo modelling.

To install:

`python pip install TopasOpt `

For help getting an appropriate environment set up, see here

The quickest way to get started is to go through the worked examples.
